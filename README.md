# 근요일

근요일은 교내 헬스장에서 더 편하게 운동 루틴을 만들고, 오늘 운동을 기록하고, 단백질 섭취와 운동 흐름을 함께 관리할 수 있도록 만든 서비스입니다.

회원가입 이후 기본 정보와 운동 목표를 설정하고, 요일별 루틴을 구성한 뒤 바로 오늘 운동 기록으로 이어지는 흐름을 중심으로 설계했습니다. 학생들이 헬스장에서 실제로 자주 쓰는 기록 경험에 맞춰 복잡한 기능보다 빠른 입력과 직관적인 확인에 집중합니다.

## 구조

이 저장소는 `pnpm` 워크스페이스 기반 모노레포입니다.

```txt
apps/
  web/    근요일 사용자 앱
  admin/  근요일 관리자 앱
```

## 실행

```bash
pnpm install
pnpm dev:web
pnpm dev:admin
pnpm build
pnpm typecheck
```

## Vercel 배포

- 사용자 앱 Root Directory: `apps/web`
- 관리자 앱 Root Directory: `apps/admin`
- 환경변수는 프로젝트별로 각각 설정

로컬에서 기존 루트 `.env`를 쓰고 있다면 `apps/web` 빌드 시 자동으로 읽도록 처리되어 있습니다.

## 주요 기능

- 회원가입과 기본 정보 설정
- 운동 목표 기반 단백질 목표 계산
- 요일별 운동 부위 및 머신 루틴 설정
- 오늘 운동 세트 기록과 달성률 확인
- 잔디 기반 운동 흐름 시각화
- 급식 및 보충 단백질 섭취 기록
